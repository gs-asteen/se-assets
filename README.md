# SE Assets

Shareable Solutions Engineering assets — dashboards, mocks, and demo HTML files. Each project is self-contained and openable in a browser with no build step.

## Catalog

### [Community Sentiment Dashboard](./community-sentiment-dashboard/)

A single-file, Horizon-branded HTML dashboard that turns 6 months of community discussion text into a single screen a community manager, moderator, or CS leader can act on. Surfaces a Sentiment Index, promoter/detractor share, an action queue of negative posts needing reply, champion members, and at-risk voices mapped to their company affiliations. Demo-ready for any conversation about community analytics or voice-of-customer.

**Live preview:** https://gs-asteen.github.io/se-assets/community-sentiment-dashboard/community-sentiment-dashboard.html

## Conventions

- Each asset lives in its own folder with its own `README.md` describing what it is, who it's for, and how to view it
- Static HTML assets should open directly in a browser with no build step
- Brand-styled work follows the **Gainsight Horizon Design System v3.1** (Noto Sans, Royal Blue 70 primary, Rich Gray neutrals, card radius 12, no gradients/emoji)
- GitHub Pages is enabled on `main` — anything in the repo is reachable at `https://gs-asteen.github.io/se-assets/<path>`
