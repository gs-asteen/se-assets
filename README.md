# SE Assets

Shareable Solutions Engineering assets — dashboards, mocks, demo HTML files, and other reusable artifacts. Open the folder for whatever you're looking for; each project has its own README.

## Contents

| Folder | What |
| --- | --- |
| [`community-sentiment-dashboard/`](./community-sentiment-dashboard/) | Single-file HTML dashboard summarizing 6 months of community sentiment for a B2B customer community. Branded with Gainsight Horizon Design System v3.1. |

## Conventions

- Each asset lives in its own folder with a `README.md` describing what it is and how to view it.
- Static HTML assets should be openable directly in a browser with no build step.
- Brand-styled work follows the Gainsight Horizon Design System v3.1 (Noto Sans, Royal Blue 70 primary, Rich Gray neutrals).
